= Andrews dotfiles (copied from Ryan Bate's Dot Files)

These are config files to set up a system the way I like it.

== Installation

  git clone git://github.com/nagi/dotfiles ~/.dotfiles
  cd ~/.dotfiles
  rake install

  git submodules init
  git submodules update

== For Bash it

Run install.sh to set up links to aliases / plugins etc.
