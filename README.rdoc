= Andrews dotfiles (copied from Ryan Bate's Dot Files)

These are config files to set up a system the way I like it.

== Installation

  git clone git://github.com/nagi/dotfiles ~/.dotfiles
  cd ~/.dotfiles
  rake install

  git submodules init
  git submodules update

== For Bash it

modify install.sh

#!/usr/bin/env bash
-BASH="$HOME/.bash_it"
+BASH="$HOME/dotfiles/bash-it"

modify bash_profile.template.bash
 
-source $BASH/bash_it.sh
+source $BASH/dotfiles/bash_it.sh

run install.sh
